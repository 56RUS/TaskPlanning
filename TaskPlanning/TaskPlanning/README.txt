Программа является клиентом к базе данных MySql. В исходниках программы имеется файл для создания базы данных со структурой, на которую расчитана программа.

Программа является планировщиком задач.
Для каждой задачи присутсвуют следующие поля:
- время постановки задачи;
- время окончания работы над задачей (если задача была закрыта);
- имя проекта, к которому относится задача;
- постановка задачи (ее описание);
- результат полученный после выполнения задачи;
- комментарии;
- номер тикета в баг-трекинговой системе, к которому имеет отношение данная задача;
- владелец / владельцы данной задачи (то есть те, кому поручено ее выполнение);
- репортер задачи (то есть тот, кто поставил данную задачу);
- текущий статус задачи (открыта, закрыта, и т.д.);
- когда и кем было произведено последнее редактирование задачи;
- общее число редактирований данной задачи.

ВАЖНО!
Каждый пользователь должен принадлежать одной из следующих групп:
- Administrator - администратор. Имеет полные права при работе с БД.
- User - рядовой пользователь.
- Guest - гостевой пользователь. Может только читать данные из БД и сохранять их в файл. Не имеет прав на добавление, удаление и редактирование задач.

Члены группы User могут редактировать и удалять только те задачи, у которых они являются либо владельцем, либо репортером.
Члены группы Administrator могут редактировать и удалять все задачи, имеющиеся в БД (даже те, у которых они не являются ни владельцем, ни репортером).

Каждый пользователь может иметь адрес email, на который будут приходить сообщения при добавлении, удалении или редактировании задачи, в которой он является репортером или владельцем.
Если у пользователя пустой адрес email, то ему не будут приходить соответствующие уведомления.
Также в коде программы возможно добавить email адрес/адреса, на которые всегда будут приходить уведомления обо всех изменениях данных в БД.


При выборке данных из БД по умолчанию происходит чтение всех задач.
С помощью соответсвующих элементов управления на форме, возмо выполнить выборочный запрос к БД.
Выборочный запрос возможен по следующим полям БД:
- время постановки задачи;
- имя проекта;
- владелец задачи;
- репортер задачи;
- текущий статус задачи.
Для выполнения выборочного запроса необходимо установить чекбокс и выбрать нужное значение.


В программе есть функция автоматического чтения данных из БД (обновления данных) через определенные промежутки времени. Для ее включения необходимо установить соответствующий чекбокс.
